# File-Transport-Management-System
Need to have:
React.js
node.js

Once files have been downloaded, open console/command line, cd to the folder with the files (should see src folder) then run:
npm install
npm start

Browser should automatically load in localhost:3000

Release Notes:
Email and password requirement checks work on the sign up, login, and forgot password pages.
No functionality to the website just yet as it has not been connected to a database or backend.
